Test Instructions:

Step 1: Email Verification

  • Enter a valid email address (e.g., test@example.com)
  • Try submitting with an empty field (should show error)
  • Try submitting with an invalid email (should show error)
  • Submit with a valid email to proceed to Step 2

Step 2: OTP Verification

  • Enter a 6-digit code (any 6 digits will work for this demo)
  • Try the "Resend" link (should show countdown timer)
  • Test backspace functionality between inputs
  • Test arrow key navigation between inputs
  • Try pasting a 6-digit code (should fill all inputs)
  • Submit with a complete code to proceed to Step 3

Step 3: New Password

  • Test the password visibility toggle
  • Test password strength meter with different passwords
  • Try submitting with non-matching passwords (should show error)
  • Try submitting with a weak password (should show error)
  • Submit with a strong password to proceed to Step 4

Step 4: Success

  • Verify the success message is displayed
  • Click "Go to Login" to return to the beginning

All functionality should work smoothly without page reloads.
